h1. Texbox: A NexT theme port for Textpattern

"NexT":https://github.com/next-theme is a popular theme for the "Hexo":https://github.com/hexojs/hexo blogging framework, and "Texbox":https://github.com/eliph/Texbox provides a portation of "NexT":https://github.com/next-theme to the  "Textpattern":http://textpattern.com CMS. Thereby, the "Textile":https://textile-lang.com/ markup languague, and many other "Textpattern's":http://textpattern.com powerful features can be used, while retaining the "NexT":https://github.com/next-theme's elegant design.

h2. Installation

Texbox requires Textpattern v. 4.7 or higher, and can be installed as follows: 

# Place the entire @texbox@ folder in the @themes@ folder of your textpattern directory.
# Go to "Presentation › Themes":https://docs.textpattern.com/administration/themes-panel and import the Texbox theme from the drop-down menue.
# Go to "Presentation › Sections":https://docs.textpattern.com/administration/sections-panel and switch the theme used to "Texbox".
# Go to "Admin › Preferences":https://docs.textpattern.com/administration/preferences-panel and set @Article URL pattern@ to @/section/title@
# Go to "Presentation › Sections":https://docs.textpattern.com/administration/sections-panel, make sections @categories@, @tag@ and @archive@. Then link them to the respective pages @categories@, @tags@ and @archive@. Also link the @articles@ section to @articles@.
# Go to "Admin › Plugins":https://docs.textpattern.com/administration/plugins-panel and install the "true_tags plugin v. 3.9.2":https://forum.textpattern.com/viewtopic.php?pid=324007#p324007. Then activate the plugin.
# Add logo image and logo thumbnail from "images folder":https://github.com/eliph/Texbox/tree/main/images
