h1. Texbox: A NexT theme port for Textpattern

"NexT":https://github.com/next-theme is a popular blogging theme for the "Hexo":https://github.com/hexojs/hexo blogging framework, and "Texbox":https://github.com/eliph/Texbox provides an portation of the "NexT":https://github.com/next-theme to the  "Textpattern":http://textpattern.com CMS. Thereby, the "Textile":https://textile-lang.com/ markup languague, and meny other "Textpattern's":http://textpattern.com powerful features can be used, while retaining "NexT":https://github.com/next-theme's elegant design.

h2. Installation

Texbox works on Textpattern 4.8.x. 

# Place the entire @Texbox@ folder in the @themes@ folder of your textpattern directory.
# Go to "Presentation › Themes":https://docs.textpattern.com/administration/themes-panel and 
import the Texbox theme from the drop-down.
# Go to "Presentation › Sections":https://docs.textpattern.com/administration/sections-panel 
and switch the theme used to "Texbox".
# Install the "true_tags":https://www.rainskit.com/reference/537/tru_tags-releases plugin
